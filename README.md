# L'Art du Style - Hair Salon Website

A modern, responsive, single-page website for the luxury hair salon "L'Art du Style".

## Features

### Complete Sections
1. **Header Navigation**
   - Fixed navbar with logo and navigation links (Accueil, Services, Galerie, Contact)
   - Functional "Prendre RDV" (Book Appointment) CTA button
   - Mobile-responsive hamburger menu with smooth slide-out animation

2. **Hero Section**
   - Full-screen background image with elegant overlay
   - Prominent headline and subheadline
   - Primary CTA button with smooth animations

3. **About Section**
   - Two-column responsive layout
   - Salon description and high-quality imagery
   - Elegant typography with Playfair Display font

4. **Services & Pricing Section**
   - Grid layout showcasing 6 main services:
     - Coupe Femme (45€)
     - Coupe Homme (35€)
     - Coloration (80€)
     - Mèches (95€)
     - Taille Barbe (25€)
     - Soin Capillaire (40€)
   - Each service card includes icon, description, and pricing

5. **Gallery Section**
   - Grid layout with 6 high-quality placeholder images
   - Hover effects with image zoom transitions
   - Responsive masonry-style layout

6. **Testimonials Section**
   - 3 client testimonials with photos and names
   - 5-star rating display using FontAwesome
   - Grid layout adapting to mobile screens

7. **Footer / Contact Section**
   - Complete contact information (address, phone, email)
   - Opening hours display
   - Social media icons (Facebook, Instagram, Twitter, Pinterest)
   - Additional CTA button for appointments
   - Copyright information

### Design & Technology

**Design Theme:**
- Minimalist and luxurious aesthetic
- Color palette: Dark grey (#333), gold/beige accents (#C9A961), white background
- Professional and high-end appearance

**Technologies Used:**
- HTML5 with semantic markup
- Tailwind CSS (via CDN) for responsive styling
- Custom CSS for advanced animations and effects
- Vanilla JavaScript for interactivity
- FontAwesome 6.4.0 for icons
- Google Fonts (Playfair Display for headings, Lato for body text)

**Interactive Features:**
- Smooth scrolling navigation
- Mobile menu toggle with icon animation
- Scroll-triggered fade-in animations using Intersection Observer
- Hover effects on cards, buttons, and gallery items
- Active navigation link highlighting on scroll
- Custom scrollbar styling

### Responsive Design
- Fully responsive across all device sizes
- Mobile-first approach
- Breakpoints optimized for tablets and mobile devices
- Fixed header that adapts on scroll
- Touch-friendly navigation and buttons

## Usage

Simply open `index.html` in a modern web browser. All dependencies are loaded via CDN:
- Tailwind CSS
- FontAwesome Icons
- Google Fonts
- Placeholder images from Unsplash

## Browser Compatibility

Works on all modern browsers:
- Chrome/Edge (latest)
- Firefox (latest)
- Safari (latest)
- Opera (latest)
